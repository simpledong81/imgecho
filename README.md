# ImgEcho

一个现代化的图片信息展示与编辑工具，支持中英文双语界面。

## 主要功能

- 📸 图片上传与实时预览
- 📝 相机参数和元数据编辑
- 🎨 自定义文字样式和位置
- 🌐 中英文界面切换
- 💾 高质量图片导出

## 快速开始

```bash
# 在线使用
https://jeffreyforest.github.io/imgecho

# 本地运行
python -m http.server 8000
```

## 技术栈

- HTML5 Canvas + JavaScript
- 纯前端实现，无需后端
- 响应式设计
